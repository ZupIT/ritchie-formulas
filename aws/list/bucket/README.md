# Description

This formula will list your buckets on aws providing only one input:

- AWS region i.e., us-east-1, us-west-1, sa-east-1

## Command

```bash
rit aws list bucket
```

## Requirements

- Golang
- Set AWS credentials (\$ rit set credentials) with ACCESS KEY ID and SECRET ACCESS KEY.
