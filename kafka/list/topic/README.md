# Kafka List topic

## Premisse

Have an available Kafka URL.

## Command

```bash
rit kafka list topic
```

```bash
rit kafka list topic --docker
```

## Description

This formula list a topic on the inform kafka URL.
You can test it using docker with the `rit docker generate compose` formula.
